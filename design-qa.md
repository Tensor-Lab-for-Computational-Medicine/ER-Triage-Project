## Residual Notes

- The implementation is a deployable frontend UI pass, not a clinical validation, regulatory approval, model safety certification, or institutional rollout sign-off.
- The primary Flowboard bundle is code-split from the legacy simulator, but the production build still reports large lazy legacy/runtime chunks (`LegacySimulatorApp`, `kokoro`, `transformers.web`, and PDF/wasm assets). Keep those behind optional paths and monitor bundle budgets before broad rollout.
- Before real worldwide medical-education deployment, run the repository readiness, learner-safety, accessibility, privacy, and clinical-review gates with institution-approved cases and faculty review.
